## 📌 Project Description
This project demonstrates the use of open source tools and basic shell scripting. It includes multiple scripts that analyze system information, inspect software, and generate a personal open source manifesto.

---

## ⚙️ Scripts

### Script 1: System Identity Report
Displays system details like kernel, user, uptime, and OS info.

### Script 2: FOSS Package Inspector
Checks if Git is installed and shows its details.

### Script 3: Disk and Permission Auditor
Analyzes system directories for size and permissions.

### Script 4: Log File Analyzer
Counts occurrences of keywords in log files.

### Script 5: Open Source Manifesto Generator
Generates a personal open source philosophy file.
